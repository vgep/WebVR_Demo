# WebVR Demo
Web VR Application with Three.js

To install and start the master branch:

```bash
git clone https://github.com/vgep/WebVR_Demo.git
cd WebVR_Demo
npm install
npm start
```

Start web browser and visit http://127.0.0.1:8080
